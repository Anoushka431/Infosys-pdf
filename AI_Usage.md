# AI Usage Documentation

## Purpose of AI Usage

Artificial Intelligence (AI) tools were used to assist during the development of this project. AI was primarily used for code assistance, debugging, documentation, and understanding PDF content extraction techniques. All code was reviewed, tested, and validated before execution.

# Areas Where AI Was Used

* Understanding PDF text extraction techniques.
* Extracting tables from PDF documents using **pdfplumber**.
* Extracting embedded images using **PyMuPDF (fitz)**.
* Converting extracted JPX images into JPEG format using **Pillow**.
* Debugging Python errors and resolving file path issues.
* Improving code structure and readability.
* Explaining Python libraries and their implementation.
* Preparing project documentation and GitHub README.
* Creating the project **requirements.txt** file.
* Git and GitHub guidance for project upload.
